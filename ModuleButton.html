Follow these rules in the order they are listed. Perform the first action that applies:

  1. Blue 'ABORT' : hold
  2. 'DETONATE' AND (#batteries >= 2) : press and release
  3. White AND indicator 'CAR' : hold
  4. (#batteries >= 3) AND indicator 'FRK' : press and release
  5. Red 'HOLD' : press and release
  6. else : hold

Releasing a Held Button

If you start holding the button down, a colored strip will light up on the right side of the module. Based on its color you must release the button when the timer shows the given number in any position:

  • Yellow strip: Release on 5.
  • Blue strip: Release on 4.
  • Else: Release on 1.
